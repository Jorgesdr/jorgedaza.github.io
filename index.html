Affiliate Program Web App

This is a live application, so the code is not publicly available. For the front end, I used React along with Zustand for global state management and Context API for handling less frequently changing states. On the backend, I implemented a server using Node.js and Express.js, adhering to a design pattern based on Controllers, Routes, and Models.

Aplicación Web del Programa de Afiliados

Se trata de una aplicación en funcionamiento, por lo que el código no está disponible públicamente. En el front end, utilicé React junto con Zustand para la gestión de estados globales y Context API para manejar estados que cambian con menor frecuencia. En el backend, implementé un servidor utilizando Node.js y Express.js, siguiendo un patrón de diseño basado en Controladores, Rutas y Modelos.
